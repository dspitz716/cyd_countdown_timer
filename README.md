# Countdown Timer
![20260310_221835](https://github.com/user-attachments/assets/c3797f42-92bb-41dc-b9e2-8d80b1831b69)


## Description:
A countdown timer created for an ESP32 cyd
- :white_check_mark: Standard Set of Holidays
- :white_check_mark: Ability to add custom date to track
- :white_check_mark: Stores custom dates in memory
- :white_check_mark: Connects to ntp server to maintain date time
- :white_check_mark: Configure wifi on boot (change from countdown menu)

## Display:
I included my User_Setup.h for the cyd display. Follow this guide if you have not setup one before https://randomnerdtutorials.com/cheap-yellow-display-esp32-2432s028r/
